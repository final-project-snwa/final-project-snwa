# ⚽ SNWA 프로젝트 개발 계획서

## 1. 프로젝트 개요
**SNWA (Sports News World Wide)**는 언어의 장벽 없이 전 세계 스포츠 소식을 가장 빠르게 접할 수 있는 플랫폼입니다. 해외 유명 스포츠 언론사의 기사를 실시간으로 수집하여, AI 기술을 통해 한국어로 번역 및 요약된 정보를 제공합니다. 단순 번역을 넘어 핵심 내용을 빠르게 파악할 수 있는 요약 서비스와 키워드 추출 기능을 통해 사용자 경험을 극대화합니다.

## 2. 기술 스택

| 구분 | 기술 스택                                           | 비고                          |
| :--- |:------------------------------------------------|:----------------------------|
| **Backend** | Spring Boot 3.4, JPA, QueryDSL                  | 안정적인 데이터 처리 및 복잡한 조회 쿼리 최적화 |
| **Frontend** | React, TypeScript, Radix UI, Tailwind CSS       | 직관적인 UI 및 반응형 디자인           |
| **Database** | MySQL 8.0                                       | 기사 및 사용자 데이터 관리             |
| **Security** | Spring Security, JWT                            | 보안 토큰 기반의 무상태(Stateless) 인증 |
| **Crawling** | Jsoup                                 |  웹페이지 데이터 수집                |
| **AI & NLP** | Spring AI (Gemini), DeepL API                   | 고성능 기계 번역 및 LLM 기반 기사 요약    |
| **Infra** | Docker, Nginx, AWS S3                           | 컨테이너 기반 배포 및 미디어 스토리지       |
| **Monitoring** | Prometheus, Grafana                             | 시스템 리소스 및 애플리케이션 모니터링       |

## 3. 핵심 시스템 아키텍처 및 흐름

### 3.1 뉴스 데이터 파이프라인 (Crawling & AI)
- **전략 패턴 크롤링**: `EspnCrawlingStrategy`, `SkySportsCrawlingStrategy` 등 매체별로 최적화된 크롤링 전략을 적용하여 확장성을 확보했습니다.
- **AI 기반 후처리**: 수집된 기사는 트랜잭션 커밋 후 비동기 이벤트로 처리되며, **DeepL**을 통한 고품질 번역과 **Gemini**를 이용한 3줄 요약 및 태그 생성이 자동으로 수행됩니다.
- **스케줄링**: Spring Scheduler를 통해 주기적으로 최신 뉴스를 업데이트하여 실시간성을 보장합니다.

### 3.2 계정 및 권한 보안 (Auth & Security)
- **JWT 인증**: Access/Refresh Token 메커니즘을 도입하여 보안성과 사용자 편의성을 모두 잡았습니다.
- **RBAC**: 일반 사용자(`USER`)와 관리자(`ADMIN`) 권한을 철저히 분리하여, 관리자 페이지 접근 및 크롤링 제어는 인가된 관리자만 가능합니다.
- **CORS & CSRF**: 프론트엔드와의 안전한 통신을 위한 CORS 설정 및 불필요한 CSRF 비활성화를 통해 REST API 환경에 최적화했습니다.

### 3.3 사용자 경험 및 커뮤니티 (UX & Community)
- **반응형 검색**: QueryDSL을 활용한 다조건 검색(제목, 내용, 카테고리 등)으로 원하는 기사를 빠르게 찾을 수 있습니다.
- **클릭 로그 분석**: 사용자가 어떤 카테고리의 기사를 많이 보는지 클릭 로그를 수집하여, 향후 개인화 추천의 기반 데이터를 마련했습니다.
- **관리자 대시보드**: 기사 관리, 사용자 관리, 그리고 크롤링 작업의 상태를 한눈에 모니터링하고 제어할 수 있는 직관적인 대시보드를 제공합니다.

## 4. 향후 확장 계획
- **개인화 추천 시스템**: 수집된 사용자 클릭 로그와 관심사를 분석하여 맞춤형 뉴스 피드 제공.
- **알림 서비스**: 관심 키워드나 좋아하는 팀의 새 기사가 올라오면 실시간 푸시 알림 전송.
- **다국어 지원 확장**: 현재 영-한 번역을 넘어 다양한 언어권의 뉴스 수집 및 번역 서비스로 확대.
